Nightpigeon Documentation

Nightpigeon is a plugin-based Discord moderation bot inspired by Zeppelin. Each feature is a plugin that can be independently enabled and configured via YAML in the dashboard.

Configuration Format

Each guild has a YAML config file managed through the dashboard. The top-level structure is:

prefix: '!'

levels:
  users:
    "123456789012345678": 100   # user ID: level
  roles:
    "987654321098765432": 50    # role ID: level

plugins:
  moderation:
    enabled: true
    config:
      mute_role: 123456789012345678
      dm_on_action: true

All configuration is hot-loaded — no bot restart needed after saving.

Permissions (Levels)

Nightpigeon uses a numeric level system (0–100), plus level 1000 for the bot owner.

LevelDescription
0Regular users (no permissions)
25Junior mods — warn, view cases
50Moderators — kick, mute
60Senior mods — ban
75Admins — delete cases
100Server owner / full access
1000Bot owner

Automod Plugin

13 built-in rules. Each rule supports custom actions: delete, dm_user, warn, mute, kick, ban.

automod:
  enabled: true
  ignore_roles: [123456789]
  ignore_channels: [123456789]
  rules:
    spam:
      enabled: true
      max_messages: 5
      per_seconds: 4
      action:
        delete: true
        warn: true
    bad_words:
      enabled: true
      words: ["word1", "word2", "wild*card"]
      action:
        delete: true
        dm_user: true
      message: "Your message was removed for containing a prohibited word."
    invite_links:
      enabled: true
      allow_own_server: true
      action:
        delete: true
    caps_spam:
      enabled: true
      min_length: 10
      max_caps_percent: 70
      action:
        delete: true

Available rules: spam, bad_words, invite_links, mass_mentions, caps_spam, link_spam, repeated_text, mass_emojis, zalgo, newline_spam, attachment_spam, mention_everyone_attempt, new_account_gate

Commands
